# -RE-SOURCES-RELATIONNELLES

## Prérequis :

- Node.js
- npm

## Dépendances

Pour installer les dépendances, exécutez les commandes suivantes :

\```
$ npm install -g @angular/cli
\```

\```
$ npm install -g @ionic/cli
\```

## Compilation

Pour compiler le projet, exécutez les commandes suivantes :

\```
$ docker-compose up
\```
